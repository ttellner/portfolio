# Agent projects use the portfolio root requirements.txt.
#
# Simulation mode (default): no env vars required.
#
# Invite-only live Ollama via Tailscale (Option B — Railway Variables):
#   TS_AUTHKEY=tskey-auth-...          # reusable Tailscale auth key
#   OLLAMA_HOST=http://100.x.x.x:11434 # PC Tailscale IP (tailscale ip -4)
#   OLLAMA_MODEL=deepseek-v2:16b
#   OLLAMA_INVITE_PASSWORD=...         # Streamlit UI unlock password
#   TS_HOSTNAME=railway-portfolio      # optional
#
# On the Windows PC running Ollama:
#   User env OLLAMA_HOST=0.0.0.0:11434  (colon before port)
#   Tailscale connected; Ollama restarted
#
# Users enter OLLAMA_INVITE_PASSWORD in the Streamlit UI to unlock live Ollama.
# Never commit secrets — set them only in Railway Dashboard → Variables.
